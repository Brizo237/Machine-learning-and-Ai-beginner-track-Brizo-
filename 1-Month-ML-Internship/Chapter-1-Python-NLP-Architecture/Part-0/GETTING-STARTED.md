# Getting Started with Part-0

## What is Part-0?
Part-0 teaches foundational Python for machine learning. It's designed for absolute beginners.

## Quick Start (30 seconds)
1. Start with [README.md](README.md)
2. Read [01-Variables-Data-Types.md](01-Variables-Data-Types.md)
3. Run [Code_Examples/01-Variables-Examples.py](Code_Examples/01-Variables-Examples.py)
