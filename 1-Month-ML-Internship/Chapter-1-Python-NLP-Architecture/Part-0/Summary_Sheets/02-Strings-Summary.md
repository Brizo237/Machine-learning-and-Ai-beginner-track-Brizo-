# Strings - Summary Sheet

**Strings** are sequences of characters used for text.

## Key Operations
\`\`\`python
text[0]           # First character
text[-1]          # Last character
text[0:3]         # Slice (0,1,2)
text.lower()      # Make lowercase
text.upper()      # Make uppercase
text.strip()      # Remove whitespace
text.split()      # Break into list
" ".join(lst)     # Combine into string
text.replace(old, new)
\`\`\`

## Key Facts
- Strings are IMMUTABLE (can't change in place)
- Use f-strings for formatting
- Index starts at 0
- [::-1] reverses
